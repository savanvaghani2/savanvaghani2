## Frontend principles

**Clarity**  
Make complex workflows easy to understand.

**Speed**  
Keep pages fast, responsive, and smooth.

**Structure**  
Build reusable frontend systems, not one-off screens.

**UX states**  
Handle loading, empty, error, success, and permission states properly.

**Product thinking**  
Focus on what users need to do next.
